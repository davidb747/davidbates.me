# C++ 常见符号

```c++
#include <iostream>

using namespace std;
/**
 * C++ 常见符号
 */
int main() {
    int a = 1, b = 2, c;
    c = a + b;
    cout << c << endl;
    return 0;
}
```

# C++ 基本数据类型

```c++
#include <iostream>

using namespace std;
/**
 * C++ 基本数据类型 (整型的实际数据长度与机器字长有关）
 * short:
 * signed short int: 16 bit
 * [1000 0000 0000 0000 ~ 0111 1111 1111 1111]
 * 
 * int:
 * signed int: 32 bit
 * 
 * long:
 * signed long int: 32/64 bit
 * 
 * unsigned short:
 * [0 ~ 2^16 - 1]
 * 
 * unsigned:
 * unsigned int
 * 
 * unsigned long:
 * unsigned long int
 * 
 * 浮点型：
 * float
 * double
 * 
 * 字符型：
 * char:
 * unsigned char
 * signed char
 * 实质是占一个字节的整型数
 * 
 * 布尔型：(C++ 扩展出来的)
 * bool: 值为 true 和 false
 * 可以当作整数型， true 一般为 1，false 为 0
 * 把其他类型的值转换为布尔值时，非零值转换为 true，零值转换为 false
 */
int main() {
    int a = 1, b = 2, c;
    c = a + b;
    cout << c << endl;
    return 0;
}
```
